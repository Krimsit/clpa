# Create Layout Project App

## Создание проекта:

    Введите в консоль clpa

    Name - Имя проекта
    Path - Путь, куда вы ходите установить проект
    Description - Описание проекта
    Author - Автор проекта

    После всего выпоните npm install

## Директория проекта:

    -app
        -css
        -images
        -js
            -app.js
        -sass
            -main.sass
        -index.html
    -dist
    -gulpfile.js
    -package.js

## Старт проекта:

    gulp watch - запуск проекта
    gulp build - сборка проекта
